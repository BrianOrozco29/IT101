Extra Credit

Five practices one should do when creating a webpage/website:

  1. Keep the most important content at the part of the page that is visible when the user arrives
  2. Always have a link to the home page accesible from any page on the website
  3. Use sans-serif style fonts to maximize readability
  4. When implementing text links, use meaningful descriptions to provide information on said link
  5. Use bulleted lists and short paragraphs to break up text, which also enhances readability

What one should avoid:

  • Limit the use of specialty fonts, like Comic Sans
  • Do NOT use dense text. Bodies of text are not very reader-friendly
  • Animated gifs, automatic sound files, and pop-up windows are very common annoyances in web design that should be avoided
  • To make pages load quickly, avoid large images and the length of the page itself
  • Do NOT use "frozen" displays for a webpage. This could cause many users to have to scroll sideways to see the full page. Because of the limitations of a common mouse, scrolling sideways is inherently inconvenient, thus annoying users greatly
